# Incident: Ransomware Attack

## Incident ID
INC-001

## Severity
Critical

## Description
Multiple files on the file server were encrypted with the .crypto extension.

## Root Cause
A phishing email executed a malicious payload.

## Resolution
- Isolated infected machine
- Restored files from backup
- Reset compromised credentials
- Updated endpoint protection

## Lessons Learned
- Enable MFA
- Improve phishing awareness
- Schedule regular backup verification
